GET /users/:username/widgets/swingometer

Returns all the details needed for displaying the Swingometer widget.

Sample Response:

{ "energy_expected": 7.3742435555556, "energy_actual": 7.4191644444444, "angle": -0.86990817147422, "hash": "63edbd06540f105cfaceaa23ce926d35", "share": false }

Error response:

Sent in the event of missing data, or some kind of unexpected error.

{ "energy_expected": 0, "energy_actual": 0, "angle": 0, "hash": "", "share": false }

Returned value explanation:

energy_expected
The user's mean historical energy usage, in kilowatt hours. Sampled from the previous 6 weeks, on the same day as the call is made (e.g. the last six Mondays if today is Monday). If insufficient information is available, it will try to compare the previous 6 days instead. Floating-point value.
energy_actual
The user's current historical energy usage, in kilowatt hours. Sampled from the last 24 hours. Floating-point value.
angle
The 'angle' of the dial pointer, measured in degrees of arc. The range of values varies between -90 (red), 0 (yellow) and +90 (green). In the AlertMe UI, these values are displayed starting with positive angles on the left (green) and ending with negative angles on the right (red). Floating-point value.
hash
A base64 encoded string that is used to identify and share a user's Swingometer. Swingometers are shared using the following URL format: "http://data.alertme.com/swingometer/individual/<HASH>.png"
share
Whether the Swingometer is public (true) or private (false). See description for hash parameter.
