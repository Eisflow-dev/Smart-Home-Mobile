GET /systemalerts

Returns the current settings for system alerts

Sample Response:

{ "failuresEmail" : "true", "failuresSMS" : "true", "warningsEmail" : "true", "warningsSMS" : "false", "nightAlerts" : "true" }

PUT /systemalerts

Updates settings for system alerts

Arguments:

failuresEmail
true
failuresSMS
true
warningsEmail
true
warningsSMS
false
nightAlerts
true

All arguments are required.

All values can be one of true or false only.

Response:

HTTP 204 Updated

Returned errors:

400 MISSING_PARAMETER
One or more parameters are missing from request
400 INVALID_PARAMETER
One of more parameters are neither true nor false
