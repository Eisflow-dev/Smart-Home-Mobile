List of available actions

  • PlaySound
  • SendSms
  • SendEmail
  • MakePhonecall
  • ControlLamp
  • ControlCamera
  • ControlSmartPlug
  • ControlIntruderAlarm
  • WriteLogEntry

These actions are only for use with the service events and device events calls.

PlaySound

Plays (or stops) a sound, which could be a doorbell, a siren or some spoken words.

Fields:

type
"PlaySound"
sound
"STOP" to stop playing, or the name of a sound to start playing
repeat
Number of times to repeat, range:1..10 or "UNTIL_CHANGED" to loop continuously.

Named sounds are

  • DOORBELL1
  • DOORBELL2
  • DOORBELL_AND_DOG
  • DOORBELL_DOG
  • EMERGENCY_SIREN
  • EMERGENCY_ENDED
  • INTRUDER_SIREN
  • INTRUDER_ARMED_TO_NIGHT
  • INTRUDER_DISARMED_FROM_NIGHT
  • INTRUDER_ARMED_TO_AWAY
  • INTRUDER_DISARMED_FROM_AWAY
  • PANIC_SIREN

SendSms

Sends an SMS (text) message - note that this uses the "mobile" number of the contact.

Fields:

type
"SendSms"
to
"me" or a contactId.
message
Plain text string, maximum length is 140 characters Latin-1/ASCII or 70 characters UTF-8.
template
(instead of message) the name of a prebuilt template.

SendEmail

Sends an email

Fields:

type
"SendEmail"
to
"me" or a contactId.
message
Plain text string, maximum 250 characters.

MakePhonecall

Phones the specified person - note this uses the "phone" number of the contact.

Fields:

type
"MakePhonecall"
to
"me" or a contactId.
message
Plain text string, maximum 250 characters.
template
(instead of message) the name of a prebuilt template.

ControlLamp

Changes the color(s) of a lamp.

Fields:

type
"ControlLamp"
style
"COLOR" or "PARTY" or "STOP" or "OFF".
color
(when style is "COLOR") a named CSS color e.g. "DarkRed" ("DARK_RED" format also allowed) or a hex RGB code e.g. "#11BB7F".
flashing
(only when style is "COLOR") "TRUE" or "FALSE" (optional, will default to "FALSE").
duration
Duration for which the new style is to be displayed (in seconds), range:1..1800 or "UNTIL_CHANGED".

ControlCamera

Fields:

type
"ControlCamera"
deviceId
The device ID of the camera to control.
recording
"STOP" or "START".
duration
(when recording is "START") Duration to record, range:1..1800 or "UNTIL_CHANGED".

Note that duration may be treated as an approximate duration under some circumstances.

ControlSmartPlug

Fields:

type
"ControlSmartPlug"
deviceId
The device ID of the SmartPlug you wish to control.
state
"ON" or "OFF".
duration
(when state is "ON") the duration for the relay to stay on (in seconds), range: 1..1800 or "UNTIL_CHANGED".

ControlIntruderAlarm

Fields:

type
"ControlIntruderAlarm"
behavior
The behavior to activate: "HOME", "AWAY", or "NIGHT".

WriteLogEntry

Fields:

type
"WriteLogEntry"
message
Plain text string to record in the hub's log, maximum 250 characters.
