GET /users/:username/contacts

List all of a user's contacts.

Response:

[ { "id": 123, "name": "Joe Bloggs", "email": "JoeBloggs@example.com", "mobile": "+447777777777", "phone": "+442222222222" } ]

POST /users/:username/contacts

Creates a new contact and returns the contactId.

Arguments:

name
The name of the contact.
email
The email address of the contact.
mobile
The mobile number of the contact.
phone
The phone number of the contact.

email, mobile & phone are each optional but at least one of them must be provided.

Response:-

HTTP 202 Created

{ "contactId": 8967762 }

GET /users/:username/contacts/:contactId

Returns the full details of a single contact.

Sample Response:

{ "id": 123, "name": "Joe Bloggs", "email": "JoeBloggs@example.com", "mobile": "+447777777777", "phone": "+442222222222" }

PUT /users/:username/contacts/:contactId

Updates a contact's details.

Arguments:

name
The name of the contact.
email
The email address of the contact.
mobile
The mobile phone number of the contact.
phone
The phone number of the contact.

email, mobile & phone are each optional but at least one of them must be provided. Empty values are permitted, and will clear the stored values if encountered.

Response:-

HTTP 204 Updated

DELETE /users/:username/contacts/:contactId

Deletes the specified contact.

Response:

HTTP 204 Deleted
